# 🎵 Music Downloader – Mobile‑Friendly Video & Audio Downloader

A lightweight web app that lets you **watch a music video** and **download it in low quality (≈2 MB)** or high quality directly to your device.  
Designed for **mobile** with a clean, responsive interface – perfect for quickly saving songs to your phone’s Gallery or Music folder.

## ✨ Features

- 📺 **Watch Video** – built‑in HTML5 player with play/pause toggle.
- 📥 **Download Options** – one‑click download buttons for:
  - **Low Quality** – compressed to ~2 MB (ideal for slow networks)
  - **High Quality** – full resolution version
- 📱 **Mobile‑First** – fully responsive UI, large touch targets.
- 🚀 **Deploy Ready** – pre‑configured for **Vercel** (serverless Flask).
- 🗂️ **User‑Controlled Save Location** – after download, the browser prompts you to save the file wherever you like (Downloads, Gallery, Music folder, etc.).

---

## 🛠️ Technologies Used

- **Backend**: Python + Flask
- **Frontend**: HTML5, CSS3, Vanilla JS
- **Deployment**: Vercel (with `vercel.json`)
